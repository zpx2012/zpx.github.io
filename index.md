## 🚀 Featured Research

### **Oak: Optimistic TCP Accelerator**
*Under Submission, 2026* I engineered a client-side throughput booster for high-loss networks by adapting the **Optimistic ACKnowledgement (Opt-ACK)** attack vector into a performance defense tool.
* **Impact:** Achieved a **12x throughput boost** on lossy transnational links.
* **Innovation:** Built as a client-side proxy requiring zero server-side modifications; handles loss recovery locally to maintain 100% data integrity while keeping the "pipe" full.

### **The Great Bottleneck of China**
*Published in ACM SIGMETRICS/POMACS, 2020* Conducted the first global-scale measurement study identifying "The Great Bottleneck"—a hidden form of censorship achieved through the intentional under-provisioning of transnational border gateways.
* **Methodology:** Large-scale IP Geolocation and BGP routing analysis to map global interference patterns.
* **Discovery:** Proved that performance degradation can be used as a deliberate policy tool to drive users toward domestic services.

### **CMPTCP: Client-side MPTCP**
Developed a transparent proxy architecture that enables **Multipath TCP** gains (reliability and throughput) when connecting to regular, non-MPTCP servers. This allows users to utilize heterogeneous routes (e.g., Wi-Fi + 5G) using only client-side logic.
